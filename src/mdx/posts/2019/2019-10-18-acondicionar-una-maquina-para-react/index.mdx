Vamos a describir los pasos para acondicionar un ambiente de trabajo para desarrollar aplicaciones bajo React.js

Primero que nada hay que tener un editor de Texto, hay muchos en el mercado pero se recomienda [Visual Studio Code](https://code.visualstudio.com). Es Open Source y tiene mucha versatilidad para el Trabajo.

Ahora que tenemos un editor de texto vamos a necesitar 2 herramientas: [Node.js](https://nodejs.org) y [Yarn](https://yarnpkg.com).

Primero instalamos node y para ver que está activo en una terminal ponemos:

```
node -v
```

Ahora vamos a ver si tenemos instalado el Node Package Manager o NPM poniendo en la terminal lo siguiente:

```
npm -v
```

Con el npm vamos a instalar el yarn poniendo lo siguiente:

```
rpm install -g yarn
```

La "G" indica que lo vamos a instalar Globalmente en la computadora. Para verificar que Yarn se haya instalado correctamente tecleamos:

```
yarn --version
```

Una buena herramienta para pruebas rápidas es Live Server para instalarlo se pone lo siguiente:

```
yarn global add live-server
```
